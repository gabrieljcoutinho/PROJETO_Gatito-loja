# Gatito Loja


Este é o repositório do site oficial da **Gatito**, uma loja especializada em produtos e cuidados para felinos. O projeto foi desenvolvido para ser uma plataforma institucional elegante, moderna e focada na experiência do dono de gato.

## Estrutura do Site

O site está dividido em seções estratégicas para facilitar a navegação do cliente:

### 1. Home (Página Inicial)
* **Banner Principal:** Destaque para promoções e lançamentos da boutique.
* **Destaques:** Exibição dos produtos mais vendidos ou serviços premium (como banho e tosa especializado).
* **Chamada para Ação:** Botões rápidos para levar o usuário diretamente à loja ou ao agendamento.

### 2. Sobre Nós
* **História:** O surgimento da Gatito e nossa paixão por gatos.
* **Nossos Valores:**
    * **Bem-estar Animal:** O conforto do gato em primeiro lugar.
    * **Qualidade:** Curadoria rigorosa de brinquedos, rações e acessórios.
    * **Transparência:** Ética em todos os nossos serviços e relações com clientes.

### 3. Contato
* **Informações de Localização:** Endereço da loja física.
* **Canais de Atendimento:** Links diretos para WhatsApp, E-mail e redes sociais.
* **Formulário de Mensagem:** Área para dúvidas, críticas ou sugestões.

## Características Técnicas

* **Design Responsivo:** Otimizado para visualização em desktops, tablets e smartphones.
* **Paleta de Cores:** Tons pastéis e modernos que remetem a conforto e sofisticação.
* **Navegação Fluida:** Menu fixo para acesso rápido a qualquer seção do site.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica das páginas.
* **CSS3:** Estilização com Flexbox/Grid e animações suaves.
* **JavaScript:** Interatividade no formulário de contato e componentes de scroll.
